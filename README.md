
# Rubik Cube Solver

A notebook-based Rubik's Cube solver project with color detection, configuration, and demo assets.

## Overview
This repository contains a collection of Jupyter notebooks for exploring cube color detection, solving logic, and video-based demonstration workflows.

## Included files
- [color_deteiction.ipynb](color_deteiction.ipynb) — color detection workflow
- [qbr.ipynb](qbr.ipynb) — solver-related notebook
- [video.ipynb](video.ipynb) — video processing notebook
- [config.ipynb](config.ipynb) — configuration notebook
- [constants.ipynb](constants.ipynb) — constants notebook
- [helpers.ipynb](helpers.ipynb) — helper utilities notebook
- [config/settings.json](config/settings.json) — project settings
- [translations/en.json](translations/en.json) — translation strings

## Getting started
1. Open the notebooks in Jupyter Notebook or Jupyter Lab.
2. Run the cells in order to inspect the detection and solving workflow.
3. Adjust the configuration files in [config/settings.json](config/settings.json) as needed.

The project currently includes notebook-based experimentation and demo assets for visualization and testing.
