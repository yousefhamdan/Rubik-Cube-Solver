
# Rubik Cube Solver

A notebook-based Rubik's Cube solver project focused on color detection, puzzle analysis, and visual demo workflows.

## Overview
This repository brings together a small set of Jupyter notebooks for exploring how a Rubik's Cube can be analyzed from images or video, how colors are detected, and how solving logic can be tested in an interactive environment. The project is intended as a research and demonstration workspace for understanding the pipeline from input capture to solving strategy.

## What this project includes
- [color_deteiction.ipynb](color_deteiction.ipynb) — notebook for color detection and image-based analysis
- [qbr.ipynb](qbr.ipynb) — notebook related to cube solving logic and experimentation
- [video.ipynb](video.ipynb) — notebook for working with video input and visualization
- [config.ipynb](config.ipynb) — notebook for configuration handling
- [constants.ipynb](constants.ipynb) — notebook containing reusable constants
- [helpers.ipynb](helpers.ipynb) — notebook with helper functions and utilities
- [config/settings.json](config/settings.json) — configuration file for the project
- [translations/en.json](translations/en.json) — language strings used by the project

## Getting started
1. Open the notebooks in Jupyter Notebook or Jupyter Lab.
2. Run the cells in order to inspect the detection and solving workflow.
3. Adjust the configuration files in [config/settings.json](config/settings.json) as needed.
4. Use the demo video asset for visual testing and comparison with the notebook output.
